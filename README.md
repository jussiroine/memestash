# MemeStash

MemeStash is a simple website for storing memes so you can quickly retrieve and share them later.

## What it does

- Create a personal stash using a short URL slug
- Upload memes (JPG, PNG, GIF, and WebP)
- Browse your stash and copy direct image links
- Delete memes you no longer need

## Tech stack

- ASP.NET Core Razor Pages
- Azure Blob Storage for meme files

## Run locally

```bash
dotnet run --project /home/runner/work/memestash/memestash/MemeStash.csproj
```
